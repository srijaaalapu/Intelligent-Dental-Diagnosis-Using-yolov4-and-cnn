# Intelligent Dental Diagnosis using YOLOv4 and CNN

## 📌 Overview
This project is an AI-based system designed to detect and classify dental diseases from X-ray images using YOLOv4 and Convolutional Neural Networks (CNN).

## 🎯 Objective
To automate dental diagnosis and assist dentists in identifying issues such as cavities, gum diseases, and tooth abnormalities.

## 🚀 Features
- Dental disease detection using YOLOv4
- Image classification using CNN
- Automated diagnosis from X-ray images
- Improved accuracy and efficiency

## 🛠️ Technologies Used
- Python
- YOLOv4
- CNN (Convolutional Neural Network)
- OpenCV
- TensorFlow / Keras

## 📂 Project Structure
- dataset/ → Dental X-ray images
- model/ → Trained models
- src/ → Source code files

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

## 📊 Output
- Detects dental problems from images
- Displays prediction results

- Improve accuracy with larger datasets

- Integrate real-time diagnosis

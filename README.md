# Intelligent Dental Diagnosis using YOLOv4 and CNN

## 📌 Description
An AI-based intelligent dental diagnosis system using YOLOv4 and CNN for detecting and classifying dental diseases from images.

## 🚀 Features
- Dental disease detection using YOLOv4
- Image classification using CNN
- Automated diagnosis system

## 🛠️ Technologies Used
- Python
- YOLOv4
- CNN
- OpenCV
- TensorFlow / Keras

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the project:
   python main.py
